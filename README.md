# Arabic Knowledge Base

A personal knowledge base designed for mastering Arabic dialects. This application combines a beautiful, modern interface with AI-powered insights to help you organize, understand, and retain vocabulary across different Arabic regions.

## Key Features

- **Multi-Dialect Support**: Seamlessly manage words and phrases across dialects like Levantine, Egyptian, Gulf, and MSA.
- **AI Enrichment**: Get instant translations, cultural context, and example usage powered by advanced LLMs.
- **Smart Organization**: Categorize entries by topic (Food, Travel, Emotions) and type (Word, Phrase, Idiom).
- **Interactive Learning**: Browse entries with a pill-based UI that expands for details, and explore connections between dialects.
- **Cloud Sync**: Secure data persistence powered by Supabase.
- **Modern UI**: A polished, responsive experience with dark mode, glassmorphism, and smooth animations.
